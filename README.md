# 📝 React Todo App

A simple and responsive Todo App built using React.js and Vite.

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 💾 Data saved using Local Storage
- 📊 Total, Completed and Pending task counter
- 🧹 Clear all tasks

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- Vite
- Local Storage
- Git & GitHub

## 📂 Installation

bash
git clone https://github.com/ashahadat7002-coder/todo-app.git

cd todo-app

npm install

npm run dev
